# Code

Contains SQL queries, analyses, and solutions for the baseball database.

## Files

| File | Description |
|------|-------------|
| `school_data_analysis.sql` | EDA and queries for school-related data |
| `salary_data_analysis.sql` | EDA and queries for salary data |
| `player_career_analysis.sql` | EDA and queries for player career data |
| `player_comparison_analysis.sql` | EDA and queries for player comparisons |

## Usage

Run any file in MySQL:

```sql
SOURCE code/filename.sql;
```
